Head and Body

The head of a site includes information for the browser. The information in the head is not seen by the user of the site. It includes keywords for search engines, the title of the website, browser information etc.Below we will create a header of our own for our new website. The body of the site includes all of the actual site content, including paragraphs, images, audio, animations, backgrounds, and so on.

Tool tips allow the user to hover over elements of a page for further explanation of what they do. For example, we would probably want a tool tip added to the link to the Star Wars page, because the link text itself doesn't give much information about what the link does.....
