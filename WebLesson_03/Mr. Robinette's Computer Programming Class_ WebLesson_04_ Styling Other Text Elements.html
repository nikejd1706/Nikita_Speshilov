Styling Other Text Elements

You should begin this lesson by reviewing our previous work on the CSS for web pages and ensure that you have a full and complete understanding so far. 


text-align

Things like title banners and the like may need to be centered on your page. Other elements will need to be centered inside of other HTML elements. For this we use text-align in your style sheet. Watch the video below and conduct your own research on text-align. 

Other Text Attributes

Border and Styles

Text, like any other tag or object in HTML, text is contained inside of a box that has a border property. Changing the border on a text box or any other HTML element requires the use of three properties...

     border-color: <color>
     border-width: <#pixels>
     border-style: <style: dashed, >
     border-top-color: <color>
     border-top-style: <dotted, dashed, etc..>
     border-top-width: <#pixels>
